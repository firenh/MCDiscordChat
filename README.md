More information + Docs: [MCDiscordChat Docs | Xujiayao's Blog](https://blog.xujiayao.top/posts/4ba0a17a/)

## Changes in FireNH's Fork from Xujiayao's original

- Players can modify the profile picture displayed in Discord by using the `/mcdcpfp set <pfp_url>` command. This can be cleared using the `/mcdcpfp clear` command.
